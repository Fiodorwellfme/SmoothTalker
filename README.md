If your flatmates/special one are tired of you yelling "Grenade!", "One down!" or "Reloading!" then this is the mod for you.  

## What does it do ?:
Heavily inspired by **ssh's VocalPlayer**, this mod autoplays voicelines when your PMC performs specific actions like :  
- Killing a PMC  
- Killing a Scav  
- Reloading  
- Throwing a grenade
- Confirming one of you team-mates' ping  
- Firing with an empty chamber  
  
## Customizable:
- Cooldown between voicelines  
- What voicelines to play  
- Delay between kill and confirming the kill  
- Schizo mode if you'd like to talk to yourself or multiplayer only if you're more of the social type  

## Installation:
1) Open the downloaded zip file.  
  
2) Drag the BepInEx folder from the zip file into your SPT folder.
